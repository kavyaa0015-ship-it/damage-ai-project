# Damage AI Project

An AI-powered web application for detecting and analyzing road or object damage using Python and intelligent processing techniques.

## About the Project

The Damage AI Project is designed to automate damage analysis and estimate damage severity using AI-based logic.  
This project helps in understanding how Artificial Intelligence can be applied in real-world damage detection systems.

The application allows users to upload images, process them, and generate damage-related outputs.

---

## Features

- Damage detection system
- Automated damage size classification
- Cost estimation functionality
- Image upload support
- Database integration
- Email service integration
- Web-based interface

---

## Technologies Used

- Python
- Flask
- HTML
- CSS
- SQLite Database
- AI/ML Concepts
